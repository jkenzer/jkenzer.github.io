About this site:

I'm using Github pages with the Dinky theme. The index page is README.md, so it's the same as my Repo's homepage. I'm using the readme_index plugin to serve this page. The README.md file is rewritten every time I commit using Github actions. It triggers the update.py script to run and walk my Repo's directory looking for any markdown files ending in .md. Links to those files are embedded in the README.md.

Constantly tinkering and learning.

I am a solutions architect for Arizona Community Foundation. I'm working to organize all our data and business processes on the Salesforce.com platform.
